# OMOI – Your AI Journal & Companion

## Overview

This project is a conversational AI assistant built using LangChain and modern LLM workflows. It integrates Retrieval-Augmented Generation (RAG), vector databases, conversational memory, and sentiment analysis to provide context-aware responses.

## Features

- Conversational AI
- Retrieval-Augmented Generation (RAG)
- ChromaDB vector database
- Conversation memory
- Sentiment analysis
- Streamlit interface

## Technologies Used

- Python
- LangChain
- ChromaDB
- Streamlit
- PyTorch
- Hugging Face Transformers

## Installation

```bash
git clone https://github.com/Pratyaksha23/OMOI_LangChain.git
cd OMOI_LangChain
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```
